# Game state save/load system

Add save/load infrastructure. On launch, show a menu screen with New Game and Load Game options. Load Game lists previous sessions with timestamp and location summary. Sessions autosave after each player action (turn/command). Store saves as JSON or similar in a user data directory. Engine must track full game state (narrative history, scene, info panel data, turn count).
