# Combat mode: h key to show hotkeys/help overlay

In tactical combat mode, pressing 'h' should display a help overlay or panel showing all available hotkeys and their actions (move, attack, end turn, retreat, etc). Currently the player has to know the keys.


## Notes

Scope change: h for help should work in BOTH combat mode and narrative/story mode, not just combat. The help overlay should show context-appropriate hotkeys for whichever mode the player is currently in (e.g. combat keys when in CombatScreen, navigation/command keys when in GameScreen).
