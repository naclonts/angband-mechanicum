# Combat: arrow keys scroll log instead of moving cursor

In combat mode, up/down arrow keys scroll the combat log instead of moving the cursor on the tactical grid. The grid panel should have default focus so arrow keys control the cursor. The combat log should only receive focus if the player explicitly tabs into it.
