# Procedural dungeon generation system

Create a dungeon/room generator for combat maps that fits within visible terminal dimensions. Generator should support variety: large rooms, small rooms, corridors, columns, mazes, etc. with terrain features like columns, water, growths, debris. Inspired by Angband's room generation approach. When entering combat from story mode, the LLM should provide guidance on room type (shape, contents, theme) which feeds into the generator parameters.
