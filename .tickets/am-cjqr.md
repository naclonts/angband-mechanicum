# Add Angband-style ctrl-move travel

In map view, holding Ctrl while moving in a direction should keep moving the player in that direction until something interesting happens. Stop conditions should include: hitting a wall or blocked tile, entering a meaningfully new/open space, spotting a monster, spotting an object or interactable, or any other event that should return control to the player. This should mirror classic Angband-style travel behavior and feel responsive from the keyboard.
